# مرصد الملاريا

لوحة وبائية عربية تعمل كصفحة ويب ثابتة. لا تُضمّن بيانات المرضى داخل المشروع؛ اختر ملف Excel من الواجهة عند كل تحديث، وتتم المعالجة محليًا داخل المتصفح.

## التشغيل

افتح `index.html` في متصفح حديث متصل بالإنترنت (لتنزيل مكتبات قراءة Excel والرسوم البيانية). ثم اختر ملف `HESN_Investigations2025.xlsx` أو نسخة محدثة منه.

## المشاركة

انشر الملفات الثلاثة `index.html` و`styles.css` و`app.js` في استضافة صفحات ثابتة مثل GitHub Pages أو Netlify أو خادم المؤسسة. لا ترفع ملف Excel إلى الاستضافة ما لم تكن لديك موافقات حماية البيانات المناسبة؛ يبقى رفع الملف من جهاز كل مستخدم هو الخيار الأكثر أمانًا.

## بنية الملف المطلوبة

تبحث اللوحة تلقائيًا عن ورقة `Sheet1` التي تحتوي حقول HESN مثل `Creation Date` و`Malaria Classification` و`ReportingOrganizationSource`. ويمكنها أيضًا اكتشاف ورقة تحتوي حقل `Investigation`.
